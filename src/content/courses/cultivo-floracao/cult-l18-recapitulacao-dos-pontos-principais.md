# Recapitulação dos pontos principais

Neste curso, abordamos tudo o que você precisa saber sobre a fase de transição e floração da cannabis. Vamos revisar brevemente os principais pontos abordados:

## Fase de Transição
- Nesta fase, as plantas de cannabis param de crescer de altura e começam a se preparar para a floração.

  
  - É importante controlar a quantidade de luz para que as plantas entrem em floração. Recomenda-se reduzir a quantidade de luz para 12 horas por dia.

  
  - A temperatura ideal para esta fase é entre 20°C e 25°C durante o dia e entre 15°C e 20°C à noite.

  
  - Neste momento, é recomendável começar a usar nutrientes específicos para a fase de floração.

  

## Fase de Floração

  
  - Nesta fase, as plantas começam a produzir buds ou flores.

  
  - A temperatura ideal para a fase de floração é entre 20°C e 28°C durante o dia e entre 15°C e 20°C à noite.

  
  - É importante manter um nível adequado de umidade para evitar problemas como o míldio.

  
  - Nutrientes específicos para a fase de floração devem ser usados para maximizar a produção de buds.

  
  - A iluminação é um fator crítico nesta fase. Recomenda-se usar lâmpadas de alta intensidade para maximizar a produção de THC.

  

## Dicas e Truques

  
  - A poda é uma técnica útil para controlar o crescimento das plantas e maximizar a produção de buds.

  
  - O treinamento de baixo stress (LST) é uma técnica simples que pode aumentar a produção de buds em até 40%.

  
  - É importante manter a limpeza da área de cultivo para evitar infestações de pragas e doenças.

  
  - A colheita deve ser realizada no momento adequado para maximizar o conteúdo de THC.
